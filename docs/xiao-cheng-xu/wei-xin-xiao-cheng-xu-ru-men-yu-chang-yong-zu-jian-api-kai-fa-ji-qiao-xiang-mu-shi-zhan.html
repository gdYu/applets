  • 移动设备分辨率与rpx

    • pt 逻辑分辨率
      • pt的大小和屏幕尺寸有关系。长度和视觉单位。
    • px 物理分辨率
    • pt包含px
  • 移动设备的分辨率与rpx（ip6 => iphone6）
    • 以ip6的物理像素750*1334为视觉稿进行设计，而在小程序中使用rpx为单位
    • ip6下1px = 1rpx = 0.5pt
    • 使用rpx，小程序会自动在不同的分辨率下进行转换，而使用为单位则不会
    • 为什么要用iphone6的物理分辨率来做设计图
      • ip6下1px = 1rpx
      • ip6 plus下1px = 0.6rpx
  • 标签
    • text
      • 文本不用text标签也能显示
      • 小程序中，只有用text标签包裹的文字，才能长按选中
