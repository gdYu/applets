GitBook部署至Github

1、github创建项目 applets（必须同时创建readme）

2、clone至gitbook存储目录的Import文件夹下

3、通过gitbook editor打开，编辑保存

4、gitbook build

5、保存并提交代码

6、创建新分支gh-pages，将_book文件夹中内容复制出来，删除其它所有文件

7、提交至gh-pages分支

8、https://gdyu.github.io/applets/
